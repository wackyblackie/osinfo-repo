OSinfo

OSinfo is a tool written in bash using zenity, it is designed to be like Neofetch. I worked hard on making a deb for it and distributing it to te web.

File Tree 
  ..
  |-- deb
  |       |-- amd64
  |               |-- osinfo_1.0-1ubuntu1_amd64.deb
  |       |-- InRelese
  |       |-- KEY.gpg
  |       |-- Packages
  |       |-- Packages.gz
  |       |-- Release
  |       |-- Release.gpg
  |-- docs
  |       |-- index.html
  • deb is the main deb folder. It contains all the files needed to be an apt server.
  • deb/amd64/osinfo*.deb is the installable deb file I have not tested it. Please notify me on GitHub if you encounter any errors, or if the programme is not installed.
  • deb/Packages is basically The Control File; and deb/Packages.gpg is a GPG Signed Packages file.
  • deb/Release is Checksums of the files in deb/*. It contains MD5Sum, SHA1, SHA265, and SHA512 Checksums. deb/Release.gpg is a GPG Signed Release file.
  • docs/html is the page you see right now. It is the "startup" page.
